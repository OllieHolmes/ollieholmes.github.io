Important Questions to ask:

How long is the animation?


When is the project due?


Will there be audio / music?

Yes No

When will the audio be delivered?


Who will be the main project contact for this project?


What kind of style are you trying to achieve?


Do you have any examples?


How do you want the project delivered when it is finished?

Estimate

Write a list of animations needed for this project

Write a list of 3D assets needed to complete the animations

Estimate the number of reviews needed for each animation

Estimate the amount of time needed for each animation

Write an estimate based on details collected and send to client

Schedule

Determine how many working days you have before the project needs to be delivered:

Find the total number of hours available for 1 person working full time (eg. 8h x 10 days to complete the project = 80h)

Divide the estimated working hours by working hours for one worker (eg. 120 hours for the project / 80 hours = 1.5 employees needed)

Write a schedule on shared calendar
